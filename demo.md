# Windows VS2019环境下的MarkDown常用语法

# 插件安装
菜单栏下，扩展->管理扩展->搜索markdown>下载安装即可
# 常用语法
* 1.标题用 # 号标记，可表示1-6 级标题，几级标题对应几个 # 号

* 2.正文直接写，如想要换行在一行写完后面空2格回车，或者直接空1行回车

* 3.几种字体：
	*斜体文本* _斜体文本_ **粗体文本** __粗体文本__ ***粗斜体文本*** ___粗斜体文本___
* 4.分割线
	在一行中用3个以上的星号、减号、底线来建立一个分隔线，行内不能有其他东西，在星号或是减号中间插入空格也可。

* 5.有序列表和无序列表
  * 无序列表使用星号(*)、加号(+)或是减号(-)作为列表标记，这些标记后面要添加一个空格，然后再填写内容,如* 1,+ 2,- 3
  * 有序列表使用数字加上 . 号并且加上空格来表示，这样回车后还能自动添加下一个列表如：1. 列表，2. 列表
	注：列表嵌套只需在子列表中的选项前面添加4个空格即可

* 6.插入代码
	行内代码：正文中的代码，通过输入`` 框住，`是英语键盘1旁边的那个键，如`print("hello wrold")`
	代码块：代码块通过两行 ``` 符号框住，如果你写的代码是某种语言，那么可以在第一行末尾加上这个语言的名字，或者代码区块使用 4 个空格或者一个制表符（Tab 键）
```C
printf("hello wrold")
print("hello wrold")
print("hello wrold")
```

* 7.插入链接
	* [链接名称](链接地址)或者<链接地址>，如[百度](https://www.baidu.com)<https://www.baidu.com>

* 8.插入图片
	* 本地图片：
	![图片](图片\xxx.jpg) ，其中[]中填名称，()里填路径，相对绝对路径都行
		* __临时方法__<img src="xxx"/>，xxx改为路径即可

	* 网络图片
	![]()，[]里面放图片名称，()里面放刚刚复制的图像链接，如
	![CSDN](https://csdnimg.cn/cdn/content-toolbar/csdn-logo_.png?v=20190924.1 "CSDN")：和上面的差不多就是()里面放图片的网址。
	
    * base64图片转码：
		* 用在线编码工具：http://kz16.top/png2base64.html
		* 打开，将图片复制，粘贴或者从本地文件上传；
		* 然后右键选择复制图像连接；
		* 若图形链接太长，将<!-- #region -->放在图像链接的上面1行，<!-- #endregion -->放在图像链接下面1行，可以看到<!-- #region -->旁边有个向下的箭头，点击就能把图像连接收起来。
